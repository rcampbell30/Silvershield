Family setup

Set up Silver Shield for safer browsing

Silver Shield gives calm scam warnings in the browser your family already uses. It checks locally in this MVP and does not send checked messages to a server.

1

Check suspicious messages

Paste an email, SMS, link, or message into the popup and get plain-English warning signs.

2

Warn on risky pages

If a page has obvious scam-like language, Silver Shield can show a small warning banner.

3

Ask a trusted person

Add a trusted contact so high-risk results can say who to ask before continuing.

📌

Important: pin Silver Shield to the toolbar

  1. Click the browser puzzle-piece/extensions icon.
  2. Find Silver Shield.
  3. Click the pin icon so it stays visible.

Pinning it makes it easier for the person you are protecting to open Silver Shield before clicking a link or sending money.
