# Silver Shield Store Listing Copy

Use this copy as the starting point for Chrome Web Store and Microsoft Edge Add-ons listings.

## Extension name

Silver Shield

## Short description

Scam warning signs for the browser your family already uses.

## One-line pitch

A privacy-first browser extension that helps users spot scam warning signs before clicking links, sending money, or sharing sensitive details.

## Long description

Silver Shield is a privacy-first scam warning extension for Chrome and Edge.

It is designed for older or vulnerable users and the family members who help protect them. Silver Shield helps users pause before clicking suspicious links, sending money, or sharing sensitive details.

You can paste suspicious messages, emails, SMS content, or links into the Silver Shield popup. The extension checks for common scam warning signs and gives a plain-English result:

- Likely safe
- Be careful
- High risk signs found

Silver Shield can also show a calm warning banner on pages with obvious scam-like language.

Key features:

- Local scam-risk checker for suspicious messages and links.
- Plain-English warning signs and next steps.
- Optional page warning banners.
- Family/trusted-contact setup.
- Trusted and blocked domain controls.
- Recent-check history with short previews only.
- Local-only privacy model in this MVP.

Silver Shield v0.1.0 has no account system, no backend, no telemetry, no payments, and no external AI/API calls. Checks run locally in the browser.

Silver Shield is a warning tool, not a guarantee. It can make mistakes. When in doubt, ask someone you trust or contact the organisation directly using a known official route.

## Review notes

Silver Shield v0.1.0 is a local-first extension. It does not call external APIs, execute remote code, collect telemetry, or require an account.

To test:

1. Install the extension package.
2. Open the Silver Shield popup.
3. Paste a fake demo message from `demo/demo.html`.
4. Click “Check for scam signs”.
5. Open Settings to review trusted contact, sensitivity, trusted domains, and blocked domains.

Fake test example:

```txt
Royal Mail: Your parcel is waiting. Pay a small redelivery fee within 24 hours to avoid return: http://royal-mail-fee.top/claim
```

Expected result: high risk signs found.

## Copy to avoid

Do not use these phrases:

- Guaranteed scam protection.
- Blocks every scam.
- Stops fraud.
- Detects all phishing.
- Definitely safe.
- Definitely a scam.
- Bank-grade security.

## Safer marketing wording

Use these phrases:

- Scam warning signs.
- Helps users pause before clicking.
- Plain-English risk checks.
- Local-first scam-risk checker.
- Can make mistakes.
- Ask someone you trust.
- Contact the organisation directly.
