Silver Shield Support

Silver Shield is a warning tool, not an emergency service. If money was sent or bank details were shared, contact your bank immediately using the number on your card or official banking app.

What Silver Shield helps with

Silver Shield helps identify scam-like warning signs in suspicious messages, links, and web pages. It is designed to help users pause and check before clicking, paying, or sharing sensitive details.

Common help topics

  • Open the extension from the browser toolbar.
  • Paste a suspicious message into the popup checker.
  • Use Settings to add a trusted contact.
  • Use trusted and blocked domains carefully.
  • Clear recent check history from the popup.
  • Reset dismissed site warnings from Settings.

Limitations

Silver Shield can make mistakes. It cannot guarantee that something is safe or definitely a scam. When in doubt, ask someone you trust or contact the organisation directly using a known official route.

Emergency steps

  • If money was sent, contact your bank immediately.
  • If a password was shared, change it through the official website or app.
  • If a verification code was shared, contact the affected service directly.
  • If remote access software was installed unexpectedly, disconnect the device from the internet and ask a trusted technical person for help.

Reporting bugs

Use the Silver Shield GitHub repository or the support contact listed in the store listing to report bugs, confusing warnings, or false positives.
